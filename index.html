Naming Variables in JavaScript

In Javascript, there are rules for variable names, here they are:

  • Variable names cannot contain spaces.
  • Names must begin with a letter, an underscore (_) or a dollar sign ($).
  • Variable names can only contain letters, numbers, underscores, or dollar signs.
  • Variable names are case-sensitive.

A variable name should accurately identify your variable's intended purpose. When you create good variable names, your JavaScript code becomes easier to understand and easier to work with, which leads to better maintainability and readability for the coder taking over from you, or even you - a week or so after you finished your coding and you're trying to remember what the variable is for.

Camel Casing

When using camel case for a variable name, you start by making the first word lowercase. Then, you capitalize the first letter of each word that follows. So, a capital letter appears at the start of a second word and at each new subsequent word that follows it.

Prefix Operator in JS

The prefix increment or decrement applies the increment or decrement to the varibale before the variable is used.

Example:

let counter =7;

let newCounter=++counter*2;

newCounter has a value of 16!

Postfix Operator in JS

The postfix increment or decrement applies the increment or decrement to the variable after it is used.

Example:

let counter =7;

let newCounter= counter++*2;

newCounter has a value of 14!

Conditional(if): Single Select Statement

A single direction is possible

Example:

if(x>y)

x=x-4;

Conditional(if): Two-way Selection Statement

2 Choices!

Example:

if (x>y)

x=x-4;

else

x=x-5;

Note that the default if the first clause is false, is the else.

Conditional(if):Multi-way Selection Statement

Multiple Choices!

Example:

if (x,y)

x=x-4;

else if(x>z)

x=x-5;

else if (x>w)

x=x-6;

else

x=x-1;

Note that the default if none of the clauses are true, is the else.

Switch Statement in JS

The switch statement is used to perform different actions based on different conditions. Don't forget the break!

Example:

switch(expression){

case x:

//code block

break;

case y;

//code block

break;

default;

//code block

}

Ternary Operator in JavaScript

This is a one-line shorthand for an if-else statement. It's called the conditional operator. Here is an example of code that could be shortened with the conditional operator:

Example:

let userType;

if (userIsYoungerThan18){

userType = "Minor";

} else {

userType = "Adult";

}

This can be shortened with ?: like so:

var userType = userIsYoungerThan18 ? "Minor" : "Adult";

For Loop, JavaScript

Loops a block of code a number of times.

While Loop, JavaScript

Loops a block of code a number of times.

Example:

Function

A block of code, given a name with which to invoke it.

Values are returned from a function with the return statement. Values are passed to a function as arguments, from the calling code. These arguments correspond to parameters in the function definition.

Example:

const adder= (x,y)=>{

return (x + y);

}

In the calling code,

let sum = adder(4,5);

sum has a value of 9.

Note that the 4,5 are the arguments, and the x,y in the definition are the parameters.

Function Declaration

Syntax is as follows

Example:

function calcRect(width, height){

return width * height;

}

Function Expression

Syntax is as follows

Example:

const calcRect = function(width, height){

return width * height;

}

Arrow Function

Syntax is as follows

Example:

const calcRect = (width, height)=>{

return width * height;

}

IIFE (Immediately Invoked Function Expression)

Syntax is as follows

Example:

let area;

(function (width, height){

area = width * height;

})();

Code is executed immediately, is not called specifically.

Objects in JavaScript

JavaScript is terms an Prototype Based Object Oriented Langauge, because you can create objects as prototypes, without having to new them up (specially allocate memory for them). Use {...} to delineate an objects, and use an attribute-value pairs.

Example:

let meal={

entrée: "Salmon and potatoes",

dessert: "Peach Cobbler",

wine: "Moscato"

}

How to Access Parts of an Object in JavaScript

Use

  1. The dot access operator
  2. Brackets notation

Example:

let meal = {

entrée: "Salmon and potatoes",

dessert: "Peach Cobbler",

wine: "Moscato"

}

... meal.wine ... is "Moscato"

Meal["wine"] is also "Moscato".
